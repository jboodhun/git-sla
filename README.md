# Git Dashboard Project

## View Demo


## Set Up

Install yarn globally:

```
npm install -g yarn
```
Use yarn to install all your modules:

```
yarn install
```

Then install the bower libraries:

```
bower install
```

To run the App:

```
yarn run start
```

To build the App:

```
yarn run build
```


## Rate Limit For Github API Calls

As an anonymous user, you are allowed 60 calls / hour.
